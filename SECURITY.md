# Security Policy

If you believe you've found a security vulnerability in this project, please follow these steps:

1. Create a private issue or email security@nexusedgesystems.example (replace with real address) with details.
2. Include steps to reproduce, impact, and any PoC code if available.
3. Do not publicly disclose the issue until it is fixed.

We will acknowledge receipt within 48 hours and work with you to resolve the issue.
